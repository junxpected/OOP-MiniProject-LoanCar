# LoanCar 

Навчальний міні-проєкт з ООП: система розумного каршерінгу, яка враховує стаж водія та автоматично розраховує запас ходу для різних типів двигунів.

## Структура проєкту (Clean Architecture)
- **LoanCar.Domain:** Бізнес-правила, сутності (`User`, абстрактний `Vehicle`), об'єкти-значення (`Location`) та патерн `Result`.
- **LoanCar.Application:** Бізнес-логіка та сервіси (`AuthService`, `RentalService`).
- **LoanCar.Infrastructure:** Збереження даних (In-Memory репозиторії).
- **LoanCar.Console:** Користувацький інтерфейс.
- **LoanCar.Tests:** Юніт-тести бізнес-логіки.

## Як запустити
1. Відкрийте термінал у кореневій папці проєкту.
2. Виконайте команду для запуску консольного інтерфейсу:
   `dotnet run --project src/LoanCar.Console`

## Як запустити тести
Виконайте команду: 
`dotnet test`
